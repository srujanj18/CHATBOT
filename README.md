# Chatbot Project

## Overview
This project is a chatbot built using the LangChain framework and integrates various tools to provide users with a conversational interface. The chatbot can answer questions, perform actions based on user input, and utilize external APIs for enhanced functionality.

## Features
- Conversational AI powered by the Llama-3.3-70b-Specdec model.
- Integration with Tavily for search capabilities.
- Dynamic message handling using LangChain's message types.
- Visual representation of the workflow and state management.

## Requirements
- Python 3.7 or higher
- Required libraries:
  - `langgraph`
  - `langchain_groq`
  - `langchain_core`
  - `langchain_community`
  - `IPython`
